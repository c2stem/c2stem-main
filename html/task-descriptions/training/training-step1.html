Before we get into the physics, the first few lessons are designed to get you comfortable with the SNAP programming environment. The more creative you are with these lessons - the easier the physics lessons will be!


  • Activity 0.1.A: Draw a Line.

    Let’s start simple. In the BUILD tab, you have a turtle (your object) and a blank canvas (your stage).

    • - Below the stage, make sure the “Turtle” object is selected.
    • - Drag and drop the [When Green Flag Clicked] block from the “Control” category on the top, left side of the BUILD page to the “Scripts” section in the middle of the screen.
    • - In our class presentation we discussed delta t and the equation for changing the position using velocity. Drag and drop [set delta t] to your program and set the delta t to 1.
    • - In order to draw the line, we will need to use the [pen down] block from the “Pen” category of blocks. Drag and drop [pen down], connecting it to the [set x position to () m] block.
    • - We want to draw a line that is 20m long. In the “Motion” category, use the [move () steps] block and insert the equation discussed. We will initially “hard code” velocity by typing the value we want the velocity to be into the equation. If we want a line that is 20m long, what value should we multiply our delta t by?
    • - Click on the green flag to run your program. When you’re finished with your program, answer the task question in the task description and submit.

  • Activity 0.1.B: Draw an Octagon.

    It is time to get more artistic! We now want our turtle to draw a triangle. We can’t simply tell the turtle “draw a triangle” and it will automatically know what to do. We have to give it step by step instructions - including the length it should travel for each side and the angle it needs to turn before drawing the next side.


    • - Follow the initial instructions described in the previous task to ensure that the correct object is selected, the green flag is added, and the pen is down. You may want to “clear” your canvas by running the [clear] block from the “Pen” category without any other code.
    • - Using the [move () steps] block from the “Motion” category with the same equation as used in the previous task, program the turtle to draw an octagon with sides 20m long.

    Hints
    • - You may need to use a loop structure from the “Control” category to repeat actions - and to simplify your code!
    • - In the “Physics” category, we have added options to set or change the angle the object is directed.
    • - By what angle does the turtle need to turn at each edge of the octagon?

  • Activity 0.1.D: Circles.

    As we move forward with the physics, you will be introduced to properties and behaviors of objects. In order to allow for our turtle to draw a circle, we have added two behavior blocks for you called [update x position by m] and [update y position by m]. Your task is to update this block with the equation necessary to update the position of the turtle (velocity times delta t).

    • - Follow the initial steps used in the previous activities (verifying object selected, adding the [When green flag clicked] block, etc).
    • - Set the x velocity and y velocity to 0.5.
    • - Make sure the pen is down prior to starting the loop!
    • - Now, in order to draw a circle, we will need to use the behavior blocks and repeat them a specific amount of times in a loop.
    • - Add the preferred loop that you would like to use for your program.
    • - In each loop, make sure to change the angle of the turtle by a certain degree.
    • - Add the two behavior blocks to your loop and insert the appropriate equation that uses velocity to update the position of the object.
    • - Run your program. What do you see? What happens if you change the values of the velocities? How about the change in angle in each loop?
