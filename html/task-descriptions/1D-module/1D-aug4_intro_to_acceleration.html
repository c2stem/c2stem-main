• Activity 1.4.A - Increasing velocity: make a prediction

    Prediction task:

    • - Using the model planning tool and the simulation blocks, make the sloth move with constant velocity to the right (as you did previously)
    • - Start at position = 0 m; End at position = 10 m; Velocity = 1 m/s
    • - Imagine the sloth will start at rest (velocity = 0), then increase its velocity by 1 m/s each second.
    • - Predict the sloth’s motion by completing Table 2 and Plot 2 in your paper worksheet. The first couple rows are done as an example.

    Hints

    • - Remember Δt = 1 second.
    • - Use the relation (position change) = (velocity change) * Δt to calculate the sloth’s new position each second.

    What is the sloth’s x velocity in m/s and x position be after 5 seconds? How about after 23 seconds?
  • Activity 1.4.B - Build your model: decreasing velocity over time

    Program description:

    • - Build a model where the sloth starts with velocity 4 m/s, and decreases its velocity by 1 m/s each second.
    • - The sloth starts at position = 20 m; Ends when the sloth reaches a velocity of 0 m/s.
    • - Use Δt = 1 second

    Run the program and use the “graphing view” buttons to answer the following questions: (1) What is the position of the sloth after 3 seconds? (2) at what time does the sloth cross the 10m mark?



  • Activity 1.4.C - Exploration and Reflection

    Exploration:

    • - Let’s run an experiment, where we try three other values of Δt, i.e., 0.01s, 0.1s, and 0.5s. Look at the distance versus time graphs. What is the difference between these graphs, and why do you see this difference?

    Try reversing the order of your change position and change velocity blocks. What happens?
