Motion describes how things change over time. Physics explores and quantifies these changes for pretty much everything in the universe--and possibly some things not in the universe. We as physicists begin with the motion of objects, or small isolated bodies or particles we assume have no internal structure. If that sounds like an oversimplification, that’s because it is, but it also makes for a good way to start modeling that motion.

When modeling motion, we as physicists want to silence that screaming feeling of loneliness from being a speck in an enormous universe, but we also want to describe and reliably reproduce changes in an object. Those changes depend on where the object is, or its position, and how that position changes. Flip books or animations both model motion. How do otherwise non-moving images model motion?
