Activity 1.4 - Build a constant velocity model

Program description:
  • Using the model planning tool and the simulation blocks, make the sloth move with constant velocity to the right (as you did previously)
  • Start at position = 0 m; End at position = 10 m; Velocity  = 1 m/s
  • Use to set your simulation step Δt = 1 second.
  • Run the program and use the “graphing view” and/or “table view” buttons to complete Table 1 and Plot 1 in your paper worksheet.

Activity 1.5 - Increasing velocity: make a prediction

Prediction task:
  • Imagine the sloth will start at rest (velocity = 0), then increase its velocity by 1 m/s each second.
  • Predict the sloth’s motion by completing Table 2 and Plot 2 in your paper worksheet. The first couple rows are done as an example.
Hints
  • Remember Δt = 1 second.
  • Use the relation (position change) = (velocity change) * Δt to calculate the sloth’s new position each second.

Activity 1.6 - Build your model: increasing velocity over time

Program description:
  • Using your constant velocity model as a starting point, add some code to make the sloth move in the way you predicted in Table 2 and Plot 2
  • The sloth starts at rest (velocity = 0), and increases its velocity by 1 m/s each second.
  • The sloth starts at position = 0 m; Ends at position = 10 m
  • Use Δt = 1 second
  • Run the program and use the “graphing view” and/or “table view” buttons to compare your prediction with your model. Do the graphs and table confirm the motion you predicted?
Hint
  • Think about how you calculated the velocity in Table 2 and Plot 2. How will your program update the velocity in each simulation step?

Activity 1.7 - Decreasing velocity: make a prediction

Prediction task:
  • Imagine the sloth will start at velocity = 4 m/s), then DECREASE its velocity by 1 m/s each second UNTIL IT STOPS.
  • Predict the sloth’s motion by completing Table 3 and Plot 3 in your paper worksheet. The first couple rows are done as an example.
Hints
  • Remember that Δt = 1 second.
  • Use the relation (position change) = (velocity change) * Δt to calculate the sloth’s new position each second.
  • How will you specify the condition for stopping the simulation?
 

Activity 1.8 - Build your model: decreasing velocity over time

Program description:
  • Using your previous model as a starting point, modify it to make the sloth move in the way you predicted in Table 3 and Plot 3
  • The sloth starts with velocity 4 m/s, and decreases its velocity by 1 m/s each second.
  • The sloth starts at position = 0 m; Ends when the sloth reaches a velocity of 0 m/s.
  • Use Δt = 1 second
  • Run the program and use the “graphing view” and/or “table view” buttons to compare your prediction with your model. Do the graphs and table confirm the motion you predicted?

Activity 1.9 - Exploration and Reflection

Exploration:
  • Explore your model by changing the code to make the sloth move in different ways. Explore different values of position, velocity, velocity change rate, and Δt. Be sure to try negative values.
  • Use the graph and table view buttons to confirm that your program works as designed.
  • Try reversing the order of your change position and change velocity blocks. What happens?
  • Answer the Reflection questions in your paper worksheet
