Activity 1.1.A - Use a loop:

Program description - When Green Flag is clicked:

  • - Use a loop structure to simulate the sloth’s motion to the right
  • - The program should update the sloth’s position at 1 second intervals
  • - Speed = 1 m/s
  • - Initial position = 0 m
  • - Final position = 10 m

Hints

  • - Change the position of the sloth on each loop
  • - Use the “Wait” block to slow down the execution of each loop
  • - How will your program make the sloth stop moving after 10 m?

Which loop structure did you use to program the sloth (if, if-else, repeat, etc) and why?



Activity 1.1.B - Adjusting the Time Step:

Program description:

  • - Add a variable “delta t” to the previous program that allows the user to specify the length of the time step
  • - The program must work for all “delta t” values greater than zero
  • - Test the program: What happens if you set delta t to a very small value? a larger value?

Make sure the [time in s] variable is selected. Set delta t to .5 and run your program. How long does it take the sloth to complete the journey?

Activity 1.1.C - Adjusting speed and direction

Program description:

  • - Add a variable “velocity” to the previous program that allows the user to specify the speed AND direction of the sloth’s movement
  • - The program must work for all velocity values (positive and negative)

Hints

  • - How does the position change of the sloth relate to its velocity and the delta t?
  • - What does negative velocity mean?

Set the velocity to 4m/s. How much time does it take for the sloth to finish?
